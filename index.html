Modern Features of Swift:

  • Implemnetation of Strings and Substrings becomes much easier and faster
  • Smart key paths for type-safe, efficient, extensible key value coding for Swift types
  • Enhancements to creating and manipulating Dictionary and Set types
  • Extends support of archival and serialization to struct and enum types and enables type-safety for serializing to external formats such as JSON and plist
  • Enforced exclusive access to memory

    • A Brief History of Swift:

    Swift programming language was first introduced at Apple’s 2014 Worldwide Developers Conference.

    “Initially, it was really just me messing around and nobody knew about it because it wasn’t anything to know about. But eventually, it got a little bit more serious […] So I started talking to my management and some of the engineers that were working on Clang, and they seemed excited about it. We got a couple people working on it part-time and I convinced my manager that it was interesting enough that we could have a couple of people work on it.” – Chris Lattner, January, 2017

    The initial reactions and reviews of swift were not great, but not terrible (2015ish). A lot of developers enjoyed certain aspects of Swift’s simplicity and flexibility. After some fine tuning, it was finally time for Swift 4.0 release. This was a significant release for Swift, as the popularity grew exponentially. “One of the reasons that Swift has become so popular over such a short period is probably the fact that it was made by Apple. When a company with such global name recognition and a multimillion army of loyal followers does something, the chances are it is doomed for success (or at least good press coverage).”

    Recursion in Swift:

    A function that calls itself is a simple but extremely useful technique called recursion. Recursion can implemented in every programming language, and is not too different in Swift. Recursion is a very powerful tool if it can be applied correctly and efficiently. Recursion also helps with condensing code.
